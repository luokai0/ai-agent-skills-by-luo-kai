# risingwave skill

Core RisingWave skill for AI coding agents. Covers connections, MCP server setup, the Source → Materialized View → Sink pipeline pattern, CDC ingestion, time windows, watermarks, and system catalog queries.

**For agents:** This skill activates automatically when you work with RisingWave — creating sources, writing materialized views, configuring sinks, or troubleshooting streaming pipelines.

**For humans:** See [SKILL.md](./SKILL.md) for the full skill content.

## Installation

```bash
npx skills add risingwavelabs/agent-skills --skill risingwave
```
