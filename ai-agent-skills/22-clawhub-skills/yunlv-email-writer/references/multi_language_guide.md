# 多语言邮件指南

## 一、各市场邮件礼仪

### 北美市场（美国/加拿大）
| 项目 | 要求 |
|-----|------|
| 称呼 | Dear Mr./Ms. [姓名] 或 Hi [名字] |
| 语气 | 专业但不僵硬，可适度幽默 |
| 时效 | 24小时内回复 |
| 签名 | 完整信息（电话、LinkedIn） |

### 德国市场
| 项目 | 要求 |
|-----|------|
| 称呼 | Sehr geehrte/r Herr/Frau [姓名] |
| 语气 | 极度正式，数据驱动 |
| 时效 | 工作时间回复即可 |
| 签名 | 正式、完整 |

### 日本市场
| 项目 | 要求 |
|-----|------|
| 称呼 | [姓] 様 或 電話で [职位] |
| 语气 | 非常正式、礼貌、谦逊 |
| 时效 | 24-48小时 |
| 注意 | 避免直接拒绝，表达含蓄 |

### 中东市场（阿拉伯）
| 项目 | 要求 |
|-----|------|
| 称呼 | Mr./Mrs. [全名] |
| 语气 | 友好，重视关系建立 |
| 时效 | 可等待回复 |
| 注意 | 不要在首封邮件中过于直接 |

### 拉美市场
| 项目 | 要求 |
|-----|------|
| 称呼 | Estimado/a [名字] |
| 语气 | 热情友好，关系导向 |
| 时效 | 相对宽松 |
| 注意 | 建立关系后再谈生意 |

## 二、邮件开头语对比

| 语言 | 正式开头 | 非正式开头 |
|-----|---------|-----------|
| 中文 | 尊敬的...，您好 | 您好，... |
| 英文 | Dear Sir/Madam, / Dear Mr. Smith, | Hi/Hello [Name], |
| 德文 | Sehr geehrte Damen und Herren, | Hallo [Name], |
| 法文 | Monsieur/Madame, | Bonjour [Prénom], |
| 西班牙文 | Estimado/a señor/a, | Hola [Nombre], |
| 葡萄牙文 | Prezado/a senhor/a, | Olá [Nome], |
| 俄文 | Уважаемый господин/госпожа, | Привет [Имя], |
| 日文 | お問い合わせ頂，感谢 |
| 韩文 | ...有限责任公司 敬启者, |

## 四、文化注意事项

### 需要避免的表达
- ❌ 中东：避免过于直接谈生意，先建立关系
- ❌ 日本：避免"No"，用"很困难"等委婉表达
- ❌ 德国：避免过多寒暄，直接切入主题
- ❌ 拉美：避免过于快速进入商业话题

### 各市场特点
- 🇺🇸 美国：简洁、直接、时间宝贵
- 🇩🇪 德国：重视数据、准时、详细
- 🇯🇵 日本：重视关系、礼仪、长期合作
- 🇧🇷 巴西：重视个人关系、友好交流
- 🇦🇪 阿联酋：尊重传统、重视信誉
