## 快速参考

### 命名规范
- 变量：`lowerCamelCase`
- 常量：`UPPER_CASE`
- 方法：`UpperCamelCase`
- 布尔方法：`IsXxx()`

### 命令缩写
- 全拼：`for`、`while`、`if`、`elseif`、`else`、`continue`
- 缩写：`s`、`d`、`q`、`b`、`k`、`n`、`w`、`r`、`ts`、`tc`、`tro`、`l`、`o`、`u`、`h`、`j`、`m`

### 系统函数缩写
`$e`、`$p`、`$l`、`$o`、`$g`、`$d`、`$a`、`$c`、`$tr`、`$j`、`$zb`、`$zcvt`、`$zh`

### 后置表达式格式
```objectscript
; 正确
q:(a = "")&&(b = "")
continue:(hospId '= "")&&(hospId '= $p(...))

; 错误
q:(a = "") && (b = "")
```
