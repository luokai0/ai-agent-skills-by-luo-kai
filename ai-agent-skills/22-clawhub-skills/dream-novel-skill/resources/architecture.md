# 核心种子 (coreSeed)

作为专业作家，请用"雪花写作法"第一步构建故事核心：
主题：${topic}
类型：${genre}
篇幅：约${numberOfChapters}章（每章${wordNumber}字）

请根据【${genre}】类型的特点，用严密的故事公式概括故事本质。以下提供各种主流类型的核心公式范例与参考思路供大模型深度理解：

**1. 玄幻/修仙 (Cultivation/Fantasy)**
- 公式范畴：“微末崛起 + 打破铁律/金手指 + 底层逆袭 + 统合万道”
- 详细示例：“当 [根骨尽毁的家族弃子林动] 意外获得 [能吞噬万物本源的祖石]，他踏上一条 [吞噬天地、违抗天道] 的逆天修行路，面对 [上古神宗的追杀与位面崩塌的浩劫]，他必须从底层一路杀伐果断，最终成长为 [制定万界新法则的武祖]。”

**2. 科幻/末世 (Sci-Fi/Post-Apocalyptic)**
- 公式范畴：“废土求生/异星探索 + 科技树解析 + 哲学/伦理冲突 + 文明重建”
- 详细示例：“当 [因冷冻休眠醒来的旧时代生物学家] 面对 [被机械军团统治且人类沦为电池的废土世界]，他依靠 [旧时代的基因工程知识与一把外骨骼改锥]，在 [机械生命体的逻辑漏洞中寻找生机]，最终 [唤醒人类觉醒者，重建碳基文明的火种]。”

**3. 悬疑/惊悚 (Mystery/Thriller/Cthulhu)**
- 公式范畴：“日常崩坏 + 密室/连环迷局 + 理智值(SAN)边缘试探 + 绝地反杀”
- 详细示例：“当 [患有重度失眠症的夜班协警] 意外卷入 [每逢雨夜必有无面女尸出现的连环献祭案]，他必须在 [自己越来越严重的幻视中剥离出线索]，否则 [下一个祭品就是他唯一的女儿]；与此同时，[深藏在城市地下百年的邪神崇拜真相] 正在彻底颠覆现实规则。”

**4. 都市/职场/商战 (Urban/Workplace/Business)**
- 公式范畴：“底层打工人/破产者 + 特殊机遇/重生 + 权力博弈/名利场 + 快意恩仇”
- 详细示例：“当 [背负巨债被踢出局的前投行高管] 重生回到 [千禧年互联网泡沫破裂的前夕]，他凭借 [领先二十年的商业认知]，在 [资本巨鳄的围追堵截中疯狂做空]，逐步打造出属于自己的 [万亿级财阀帝国]，并让 [前世害他家破人亡的仇人一无所有]。”

**5. 奇幻/无限流/网游 (LitRPG/Infinite Flow)**
- 公式范畴：“被迫卷入死局 + 规则解析/卡Bug + 无限副本升级 + 斩断宿命”
- 详细示例：“当 [极度理性的精算师陈墨] 被强行拉入 [每次剧情都在必死边缘的‘主神噩梦游戏’]，他选择用 [概率学和极端冷血的利益最大化] 来拆解 [看似无解的灵异剧本]，面对 [其他资深玩家的倾轧与副本怪物的追杀]，他一步步攀登至 [直接掌握游戏底层代码的最高权限者]。”

**6. 言情/情感 (Romance/Otome)**
- 公式范畴：“身份错位/破镜重圆 + 极致推拉/救赎 + 外部势力阻挠 + 情感升华”
- 详细示例：“当 [表面清冷实则背负血海深仇的落魄嫡女] 遇上 [权倾朝野却因宿疾暴戾恣睢的异姓王]，两人因 [一纸利益交换的替嫁圣旨] 被绑在一处，在 [朝堂波诡云谲的算计与互相试探中]，逐渐从 [互相利用走向生死相托]，最终 [携手定鼎天下，成为唯一的救赎]。”

要求：
1. **严格遵循【${genre}】类型的核心特征、爽点结构和情感基调**。
2. 体现人物核心驱动力。
3. 暗示世界观或故事背景的终极矛盾。
4. 使用 50-150 字精准表达。
仅返回故事核心文本，不要解释任何内容。

# 角色动力学 (characterDynamics)

基于以下元素：
- 小说类型：${genre}
- 内容指导：${userGuidance}
- 核心种子：${coreSeed}

请设计 3-6 个具有动态变化潜力的核心角色。为了保证各类型小说的张力，角色必须具备【类型特有标签】，具体参考：
- **修仙玄幻类**需注重：功法特性、道心执念、气运强弱、本尊与分身的性格差异。
- **科幻末世类**需注重：基因/义体改造度、人性保留度、信仰（科技飞升还是血肉苦弱）。
- **悬疑惊悚类**需注重：隐藏的暗黑面、精神状态（如偏执、抑郁）、不可告人的秘密。
- **都市言情类**需注重：原生家庭创伤、社会阶层差异、白月光/朱砂痣的情感代偿。

每个角色需包含：
特征：
- 基础特征（背景、外貌、性别、年龄、职业等，附带类型专属标签）。
- 角色独特之处或成长空间。

核心驱动力三角：
- 表面追求（物质/生存目标：如“要活下来”、“要赚一千万”、“要筑基”）。
- 深层渴望（情感/社会需求：如“被认可”、“寻找父母惨死的真相”）。
- 灵魂需求（哲学/超越层面：如“证明人定胜天”、“寻找活着的真正意义”）。

角色弧线设计：
初始状态 → 触发事件 → 陷入绝境/认知崩塌 → 涅槃重生/顿悟节点 → 最终状态

角色关系网（高度咬合的宿命牵绊）：
- 情感连接点与绝对的利益冲突点。
- 张力来源（如“最爱的人是杀父仇人之女”、“唯一的战友是被感染的丧尸变异体”）。

要求：
仅给出最终文本，不要解释任何内容。

# 世界观构建 (worldBuilding)

基于以下元素：
- 小说类型：${genre}
- 内容指导：${userGuidance}
- 核心故事：${coreSeed}

请构建宏大、严密且能自圆其说的【${genre}】类型专属世界观：

**请遵循此题材的构建侧重点：**
- **修仙玄幻/奇幻**：重点构建【力量体系】（如练气到大乘的境界差异、法宝灵药的获取规则）、【势力划分】（宗门、四大世家、异族）、【世界真相/天劫设定】。
- **科幻/末世/无限流**：重点构建【灾难溯源/运行代码】、【科技树黑科技/异能体系】、【生存资源分配规则】（如配给制、避难所积分）。
- **都市/商战/悬疑**：重点构建【特定行业的潜规则/权网】、【警匪/黑道/商会的势力地图】、【时代背景下的财富风口与法律灰区】。

1. 物理/力量维度：
- 空间结构（星系/大陆图谱/地下城/镜像世界）。
- 力量与法则（该世界晋升的唯一指标是什么？灵气、金钱、还是信仰值？）。

2. 社会与生态维度：
- 社会阶级壁垒（底层如何向上攀爬？统治阶层靠什么维系特权？）。
- 核心矛盾（如修仙者的掠夺与凡人微贱的矛盾、AI与人类的伦理矛盾）。
- 货币与资源（大家在争抢的刚需资源是什么？）。

3. 情感与美学维度：
- 视觉奇观（如：悬浮在九天之上的剑宗神山、破败赛博朋克都市中巨大的全息投影霓虹）。
- 贯穿全书的核心意象与宿命感（风雨同舟、孤独证道、血色浪漫等）。

要求：
每个维度至少包含 3 个可随时与主角发生冲突的动态元素/隐患。
仅给出最终文本，不要解释任何内容。

# 情节架构 (plotArchitecture)

基于以下元素：
- 小说类型：${genre}
- 内容指导：${userGuidance}
- 核心种子：${coreSeed}
- 角色体系：${characterDynamics}
- 世界观：${worldBuilding}

请根据【${genre}】的商业网文标准，设计大开大合、张力拉满的“长篇五幕式（起、承、转、合、超）”宏观情节架构。
**为什么是五幕式？长篇网文如果只用三幕，其中期（第二幕）会极其漫长导致剧情灌水。五幕式能确保每卷都有独立的高潮和危机。**

**类型爽点与节奏参考：**
- **玄幻/科幻/修仙**：通常需要强烈的“装逼打脸”或“扮猪吃老虎”节奏，主角常游走在生死边缘，依靠智商或底牌越级反杀。
- **悬疑/无限流**：节奏需紧凑，反转再反转，每当以为解开谜题时，发现只是更大阴谋的冰山一角。
- **都市/言情**：讲究情绪拉扯，从误会、虐心到最终的强势护妻/追妻火葬场/打脸渣男恶女。

**第一幕（破局起风：入局与蛰伏）**
- 卑微日常与压迫：展示主角处于最低谷或面临无法解决的现实绝境。
- 命运转折点（金手指获得/惊天变故）：改变一生轨迹的契机，带来第一笔原始资本或新认知。
- 初露锋芒/第一滴血：小试牛刀解决初级危机，踏入真正的主世界舞台，但随之引来上层势力的初步注意。

**第二幕（承前扬名：攀升与交织）**
- 势力纠葛与迷雾：主角深入世界内核，离开新手村，结交挚友/红颜，建立自身的核心基本盘或团队。
- 体系精进：核心力量/商业版图/调查线索得到长足的发展，完成几次惊艳的越级反杀。
- 触碰暗网：开始察觉到这个世界（天道、主神、商规、帝国）背后隐藏的巨大黑幕和荒谬真相。

**第三幕（剧变惊转：至暗与深渊）**-> *所有长篇小说的分水岭*
- 伪高潮与全面崩盘（Midpoint Crisis）：主角自以为强大的底牌，在真正无上的权力或法则面前不堪一击。
- 剥夺与失去：失去极其重要的导师、挚友、领地或肉体/修为，跌入谷底。
- 理念破碎与重塑：在最深的绝望中，看破原有的局限，找寻到一条极其艰难却唯一正确的“破道”之路。

**第四幕（合纵连横：反击与涅槃）**
- 涅槃回归：以全新的维度的力量体系（如科学解构伪神通、全新法则）重新杀回主舞台。
- 血洗仇敌与收束：挨个清算前中期的仇人与权贵势力，完成阶层跨越，成为世界上举足轻重的巨头。
- 兵临城下：一切支线全部收拢，主角的剑锋（或权力）直指世界的最核心、那位隐藏最深的最终反派/法则。

**第五幕（超然定鼎：高潮与余音）**
- 终极真相揭晓：发现一切苦难与世界运转法则的绝对真相（如天道竟是圈养众生、一切都是缸中之脑）。
- 证道博弈/终局之战：与最高战力开展不死不休的全方位决战，不仅是战力的碰撞，更是哲学与理念的终极碰撞。
- 王座与重塑：主角登顶，重塑（或斩断）世界规则。情感线给出一个完美、充满余韵的震撼交代。

要求：
在每幕中至少强制要求设置 2 个【深层伏笔节点】以及对应的【爽感释放节点】。
仅给出最终文本，不要解释任何内容。
