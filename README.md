## 🚀 How to Use a Skill

**In Claude Code** — copy a `SKILL.md` into your project:
```bash
cp ai-agent-skills/01-programming-languages/02-python-ecosystem/python-expert/SKILL.md .claude/skills/
```

**In Claude.ai** — paste the contents of any `SKILL.md` into a Project's custom instructions.

**Skill format:**
```
skill-name/
└── SKILL.md          ← frontmatter + instructions + working code examples
    references/       ← optional supporting docs
    scripts/          ← optional helper scripts
```
