## 📦 Installation

### Option 1 — Clone and copy
```bash
git clone https://github.com/luokai0/cloud-skills-by-luo-kai-.git
cp -r cloud-skills-by-luo-kai-/skills/* .agents/skills/
cp -r "cloud-skills-by-luo-kai-/skills 0.1"/* .agents/skills/
cp -r "cloud-skills-by-luo-kai-/skills 0.2"/* .agents/skills/
```

### Option 2 — Git submodule
```bash
git submodule add https://github.com/luokai0/cloud-skills-by-luo-kai-.git .agents/codingskills
```

### Option 3 — Use specific folder only
```bash
# Just finance skills
cp -r "cloud-skills-by-luo-kai-/skills 0.1"/* .agents/skills/

# Just physics skills
cp -r "cloud-skills-by-luo-kai-/skills 0.2/physics"/* .agents/skills/
```
