## What Are Agent Skills?

SKILL.md files that teach AI agents specific tasks. Works across **25+ platforms**: Claude Code · Claude.ai · Cursor · GitHub Copilot · Windsurf · Codex · Gemini CLI · Cline · Amp · Manus


## Usage

```bash
# Claude Code (project level)
cp -r <skill-folder> .claude/skills/

# Claude Code (global)
cp -r <skill-folder> ~/.claude/skills/

# Cursor
cp -r <skill-folder> .cursor/skills/

# OpenAI Codex
cp -r <skill-folder> .agents/skills/

# Any compatible agent
/learn @luokai0/<skill-name>
```
